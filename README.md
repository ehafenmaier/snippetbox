# snippetbox

`snippetbox` is the web application built throughout the book _Let's Go_ by Alex Edwards.

This repository contains the source code for the project and follows the structure from the book as the application is developed step by step.

## Project Context

- **Book:** _Let's Go_
- **Author:** Alex Edwards
- **Application Name:** `snippetbox`

## Running the Application

From the repository root, run:

```bash
go run ./cmd/web
```

By default, the web server listens on `:4000`.

Then open `http://localhost:4000` in your browser.
